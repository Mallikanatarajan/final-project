Surveillance CAPTCHA System

This project explores how CAPTCHA systems evolve into surveillance, behavioral tracking, and machine-learning data extraction systems. The experience gradually transforms from a familiar verification interface into simulated biometric analysis and identity archiving.

Enter the System

About the Project

The project critiques surveillance creep and the normalization of data collection through everyday interfaces. Inspired by systems like Palantir and predictive behavioral analysis platforms, the work imagines CAPTCHA not as security, but as a hidden training pipeline where human interaction becomes machine-learning data.

The interface intentionally shifts from recognizable image-selection tasks into increasingly invasive behavioral monitoring. Visual glitches, false biometric analysis, and database language are used to create discomfort and highlight how surveillance systems slowly expand beyond their original purpose.
